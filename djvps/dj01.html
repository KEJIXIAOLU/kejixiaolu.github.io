搭建 Vision 节点申请证书

1、安装证书工具

复制 curl https://get.acme.sh | sh; apt install socat -y || yum install socat -y; ~/.acme.sh/acme.sh --set-default-ca --server letsencrypt

2、申请证书

复制 ~/.acme.sh/acme.sh --issue -d 你的域名 --standalone -k ec-256 --force --insecure

这里需要更改你的域名，如果出现提示错误，可能是需要放行80端口。

3、安装证书

复制 ~/.acme.sh/acme.sh --install-cert -d 你的域名 --ecc --key-file /etc/x-ui/server.key --fullchain-file /etc/x-ui/server.crt

五、放行端口
放行指令是一样的，只要将端口443为任意端口就可以了。

放行 443 端口

复制 iptables -I INPUT -p tcp --dport 443 -j ACCEPT

放行 54321 端口

复制 iptables -I INPUT -p tcp --dport 54321 -j ACCEPT

六、检测端口是否被封
端口被封的原因是多方面的，目前并没有哪一种节点可以保证不被封，本期讲的这三种方式也不例外，所以如果你的节点突然无法使用了，可以用以下方式进行排查。

打开 ping.pe

输入 IP 检测 ping 可用

输入 IP:Port 检查端口是否可用

主要是看最后几个是否为绿色
